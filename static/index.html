This website provides the current COVID-19 data published daily by European Centre for Disease Prevention and Control as a Tableau Web Data Connector. It is for academic purposes only and does not aim for any commercial sense. Access the ECDC's website here and note the ECDC Copyright and Limited Reproduction Notices.


This instance is not meant to be online permanently, so please do not rely on this connector! Feel free to serve it yourself: Find the source code on GitHub.
